# KeySprout 小学生单词打字练习 — 需求说明

## 一、产品目标

打造一个类似“金山打字通”的网页单词输入练习工具，帮助用户通过“看单词、听发音、键盘输入、即时纠错”的方式练习英文拼写和打字熟练度。产品无需安装，打开网页即可使用。

## 二、目标用户

- 小学一年级至六年级、正在学习键盘指法和英文拼写的学生
- 希望提高英文键盘输入速度和准确率的用户
- 需要利用碎片时间进行短时练习的用户

## 三、核心功能（当前版本）

1. **单词练习**：2000 个单词均显示音标、词性和常用中文释义。
2. **即时输入判定**：用户输入时逐字标记正确或错误字符。
3. **单词发音**：点击扬声器按钮或按 `Tab`，通过浏览器语音合成播放美式英文发音。
4. **年级词库**：内置 2000 个高频常用英文单词，按词频、词长和拼写复杂度划分为小学一年级至六年级六个互不重复的练习池；每轮均匀随机抽取，单轮内不重复。
5. **虚拟键盘**：显示 QWERTY 键盘布局，实时高亮下一枚目标按键；实际按键按下时提供视觉反馈。
6. **指法提示**：逐字提示应该使用左手或右手的具体手指；输错时提示使用右手小指按 Backspace 删除。
7. **双手示意**：在 1920×1080 桌面布局中显示左右手五指图，目标手和目标手指随下一按键同步高亮。
8. **练习轮次**：支持每轮 10、20、30 个单词。
9. **数据统计**：实时显示当前进度、WPM 输入速度和正确率。
10. **快捷操作**：`Tab` 播放发音，`Enter` 进入下一词。
11. **练习结算**：完成一轮后展示速度、正确率、完成单词数，并可立即再练。
12. **响应式界面**：最佳显示分辨率为 1920×1080，同时兼容常见桌面端和手机端尺寸。

## 四、交互规则

- 输入与目标单词一致时判定为正确，短暂提示后自动进入下一词。
- 输入错误的字符以橙红色波浪线提示，正确字符以绿色显示。
- 单词未完成时点击“下一个”会提示先完成当前单词。
- 切换年级或词数后自动开始新一轮练习。
- 发音能力依赖浏览器的 Web Speech API；不支持时应给出明确提示。

## 五、视觉要求

- 视觉风格：复古打字机训练室 / 纸张编辑感。
- 主色：奶油纸色、深墨绿、橙红强调色。
- 强调排版、纸张纹理、清晰层级和适度动效。
- 确保文字对比度、键盘可操作性以及减少动态效果偏好兼容。

## 六、后续可扩展需求

- 用户自定义词库、CSV 导入与错词本
- 历史成绩、连续练习天数和学习趋势图
- 单词听写模式（隐藏拼写，只播放语音）
- 单词例句、英音/美音切换和真人音频
- 账号、云同步、排行榜和成就徽章
- 中文/英文打字文章模式

## 七、验收标准

- 页面打开后可立即开始练习，无需登录。
- 输入过程能正确反馈每个字符，完成单词后可进入下一词。
- 发音按钮和快捷键在支持 Web Speech API 的主流浏览器中有效。
- 切换难度、词数，统计和结算功能均能正常工作。
- 页面在常见桌面与移动尺寸下无明显溢出、遮挡或不可操作问题。
