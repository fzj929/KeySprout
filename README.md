# KeySprout

> 面向小学生的英文单词键盘与指法练习网页。

KeySprout 将英文拼写、键盘指法和单词发音结合在同一个练习界面中。学生可以按年级选择词库，在虚拟键盘和双手指法图的引导下逐字输入单词，并实时查看正确率、输入速度和练习进度。

**Type · Listen · Grow**

## 功能特点

- 小学一年级至六年级独立词库
- 内置 2000 个学生常用英文单词
- 每个单词均包含音标、词性和中文释义
- 浏览器美式英文发音
- 输入字符实时正误反馈
- QWERTY 虚拟键盘与下一按键高亮
- Three.js 3D 双手、符合真实比例的连续指节、可动关节与逐键指法提示
- 输错时提示右手小指按 `Backspace`
- 实时显示 WPM、正确率和练习进度
- 支持每轮随机抽取 10、20 或 30 个单词，单轮不重复
- 支持桌面端和移动端；1920 × 1080 桌面大屏可获得完整的双手指法训练布局

## 快速开始

项目不需要安装依赖，可直接打开，也可以选择使用静态文件服务器运行。

1. 下载或克隆本仓库：

   ```bash
   git clone https://github.com/fzj929/KeySprout.git
   ```

2. 进入项目目录，双击 `index.html`。
3. 建议使用最新版 Chrome 或 Edge 浏览器。

也可以使用任意静态文件服务器运行，例如：

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 使用方法

1. 选择学生所在年级。
2. 选择本轮练习 10、20 或 30 个单词。
3. 点击扬声器按钮听单词发音。
4. 根据虚拟键盘高亮和手指提示输入单词。
5. 输错时先按 `Backspace` 删除错误字符。
6. 正确输入完整单词后，页面会自动进入下一词；也可以按 `Enter` 立即继续。
7. 完成一轮后查看速度、正确率和完成数量。

移动设备可以正常显示响应式页面；如需练习标准键盘指法，建议连接实体键盘。

### 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Tab` | 播放当前单词发音 |
| `Enter` | 正确输入后立即进入下一词（页面也会自动跳转） |
| `Backspace` | 删除错误字符 |

## 技术说明

- 使用原生 HTML、CSS 和 JavaScript 开发
- 使用 Three.js 绘制半透明 3D 双手，并通过关节动画提示目标手指
- 使用 Web Speech API 播放英文发音
- 数据全部保存在本地文件中，练习过程无需请求后端服务
- 不收集、不上传用户的输入和练习成绩
- 支持 `prefers-reduced-motion`，减少不必要的动画

不同操作系统和浏览器提供的英文语音可能略有差异。如果浏览器不支持 Web Speech API，页面会给出提示，其他练习功能不受影响。

如浏览器不支持 WebGL，3D 双手会自动降级为 SVG 指法图，单词输入、发音和统计功能不受影响。

### 重新构建 3D 双手

仅修改 `hand3d.js` 时需要安装开发依赖并重新构建；直接使用网页无需执行这些命令：

```bash
pnpm install
pnpm build:3d
```
